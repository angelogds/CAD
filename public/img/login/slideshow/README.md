# Slideshow do login

Coloque aqui as imagens do slideshow da tela de login.

- Caminho web da pasta: `/img/login/slideshow`
- Formatos aceitos: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`, `.avif`, `.svg`
- As imagens são carregadas automaticamente em ordem alfabética do nome do arquivo.

Exemplo de nomes recomendados:

- `01-fabrica.jpg`
- `02-solda.jpg`
- `03-manutencao.jpg`
